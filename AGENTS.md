## Dev Server
- For simple web pages create an html file instead of an .md file
- When something more complex is needed for the web use React, Tailwind, Shadcn
- Do NOT run dev server commands like `npm run dev`, `bun run dev`, `yarn dev`, etc. The user will run these themselves.

## iOS 
- For iOS and mac projects, default to using SwiftUI
- Do NOT build the xcode project Unless asked directly to do so.

## Commits
- When you get to a stable stopping point or have finished the task, comment the change, commit the code locally.
- in the commit messages, do not put anything like "Co-Authored-By: Claude Opus" etc.
- When creating new files include these three lines in the header:
